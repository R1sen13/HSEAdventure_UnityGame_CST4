  
- - - HSEAdventures - - - 

  Это проект, созданный студентами ВШЭ с группы 25-КНТ4, для "Технологий программирования". Основная суть проекта заключается в создании собственной инди (independent) игры, 
основанной на культовом маскоте Высшей школы экономики - вороны. 

----------------------------------------

  Игра создана в 2D с разделением на уровни, каждый из которых представляет одну из популярных в мире игр: Mario, Flappy Bird, Doodle Jump. Основные ассеты были созданы 
самостоятельно на основе pixel-art. 
  
  При создании проекта задачи были разделены на несколько подгрупп: программирование уровней на игровом движке Unity, дизайн ассетов, персонажей + UI/UX, sound-дизайн и 
"склейка" всех уровней.
